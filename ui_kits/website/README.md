# Chimney Masters — Website UI Kit

Marketing-site kit (WordPress + Elementor site, recreated from the brand brief — no site export was provided, so layouts follow the brief's page list and the visual foundations, not a pixel source).

- `index.html` — interactive: header nav switches between screens
- `Shared.jsx` — Header, Footer, TrustBar, shared icons
- `HomeScreen.jsx` — hero, trust bar, services grid, coupon callout, reviews, dark CTA band
- `ServiceScreen.jsx` — service detail page (Chimney Sweep)
- `ContactScreen.jsx` — contact/quote form

Photography slots are gray placeholders — replace with real photos.
